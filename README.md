# 루틴머니 🪙

해야 할 일을 체크하면 용돈이 쌓이고, 지출을 기록하면 빠지는 간단한 용돈 장부 앱입니다.
서버 없이 브라우저 `localStorage`에만 데이터를 저장하는 **단일 HTML 파일** 앱이라, 별도 설치나 빌드 과정 없이 바로 사용할 수 있습니다.

## 주요 기능
- 🏠 **홈**: 오늘 할 일 체크 → 용돈 적립, 이번 달 요약 확인
- 📅 **달력**: 날짜별로 번 돈 / 쓴 돈 확인
- 💸 **지출**: 지출 기록 및 이번 달 지출 내역
- ⚙️ **설정**: 반복 루틴 관리, 시작 용돈 / 이월금 설정, 다크 모드

## 사용 방법

### 1. 바로 열어보기
`index.html` 파일을 다운로드해서 더블클릭하면 브라우저에서 바로 실행됩니다.

### 2. GitHub Pages로 배포하기
1. 이 저장소를 GitHub에 올립니다.
2. 저장소 **Settings → Pages**로 이동합니다.
3. **Source**를 `main` 브랜치, `/ (root)` 폴더로 설정합니다.
4. 잠시 후 `https://[사용자명].github.io/[저장소명]/` 주소로 접속하면 앱이 열립니다.

### 3. Vercel로 배포하기
1. GitHub 저장소를 Vercel에 연결(Import)합니다.
2. 별도 빌드 설정 없이 그대로 Deploy하면 됩니다 (정적 HTML 파일이라 빌드 명령이 필요 없습니다).

## 데이터 저장 방식
모든 데이터(루틴, 체크 기록, 지출 내역 등)는 **브라우저의 localStorage**에만 저장됩니다.
- 서버나 데이터베이스가 없어 별도 비용이 들지 않습니다.
- 단, 브라우저 캐시/사이트 데이터를 삭제하면 기록도 함께 사라지니 주의하세요.
- 기기를 바꾸면 데이터가 자동으로 동기화되지 않습니다 (기기별로 데이터가 독립적으로 저장됨).
